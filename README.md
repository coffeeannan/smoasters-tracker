# Smoasters Tracker

This is where we keep track of any bugs discovered in [Smoasters](https://smoasters.com).

**Please note the issue tracker is not for technical support questions.** This repository is the official public tracker for Smoasters.
All submitted bug reports and feature requests are listed here. Clients can monitor progress and track the roadmap.
